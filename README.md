# wordpress-sites
Конфиги сайтов WP для киберполигона

Для разворачивания сайтов на локальном хосте перейти в папку с файлами сайта (например: my-wp-site1) и выполнить docker compose up -d

Это конфигурация для запуска сайта на WordPress с использованием Docker. Проект предназначен для локальной разработки или тестирования и включает в себя все необходимые компоненты: веб-сервер с PHP, базу данных MySQL и готовый дамп данных.
Используется официальный образ wordpress:6.3-php8.2-apache.

Сайты доступны локально по адресу: http://localhost:(8080-9070). Либо из машин одной и той же подсети по IP адресу машины, где эти сайты развернуты.
Для увеличения лимитов загружаемых файлов подключен файл uploads.ini (увеличивает upload_max_filesize до 256 МБ и memory_limit до 512 МБ).

Содержимое: Демо-сайт с одной тестовой записью ("Hello world!"), одной страницей ("Sample Page"), стандартным комментарием и темой "Twenty Twenty-Five". Пользователь: admin Пароль: admin.
Для настройки содержимого сайтов необходимо перейти в админ панель и оттуда выполнять настройку

Так же в проект добвлена конфигурация Nginx для обращения к сайтам по доменному имени при использовании DNS сервера в сети
